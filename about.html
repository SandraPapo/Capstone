About

With recent technological advances greenhouses are becoming automated. This automation requires reliable, consistent, and quick monitoring of variables throughout the greenhouse environment. With all of the data that's being collected every day from the greenhouse there is the potential for trend analysis and optimization discovery through analysis of the data collected over time. Greenhouse pest control methods have changed to remove pesticide use, these methods require accurate tracking of the number and type of insects within the greenhouse. Currently this tracking is all done by hand and is very labour intensive.

What is the Internet of Things?

The Internet of Things (IoT) is the network of physical devices, vehicles, home appliances, and other items embedded with electronics, software, sensors, actuators, and other forms of connectivity that enables these devices to communicate and exchange data with each other, as well as more complex devices. This creates opportunities for direct integration of the physical world with computer-based systems, resulting in efficiency improvements, economic benefits, and reduced human exertions.

What is Amazon Web Services?

Amazon Web Services (AWS) is a cloud services platform that provides services to individuals, companies, and governments. Subscribers are given access to a virtual cluster of computers, available all the time, through the Internet. As of 2017, AWS owned a dominant 34% of the cloud networking market.

Bluetooth LE Sensor Network

The broadcast network topology is used for establishing one-to-many device connections. This communication structure that the sensors in the IOP system use, each sensor is connected to the gateway using the BlueTooth Low Energy (BLE) protocol. BLE devices communicate through broadcasting advertised packets. Three separate channels (frequencies) are used to reduce interference between devices. An advertising device sends a packet on one of the channels, with a random delay up to 10ms to further reduce collisions. The scanning device listens to each channel for a specific scan interval.

Gateway

The gateway connects to the cloud, and pulls the data from the sensors, formatting it before sending it to the cloud.

Cloud Services

Amazon Web Services including IoT Core, Lambda, DynamoDB, APIGateway, S3 and QuickSight was used to connect devices, add functionality, store and analyze data, and visualize business insights. AWS takes the information from the gateway and pushes it to the internal Dynamo Data Base for storage. AWS Lambda facilitates the creation of rules used to control the greenhouse based on the received information.

Website Design

The website was created so that users can access information about each zone of the greenhouse. It displays the real-time greenhouse conditions using JavaScript to fetch data from the API Gateway. Bootstrap was used for the front-end web framework, creating the cross platform, scalable, user-friendly display.

Insect Tracking

Greenhouses currently use kilometers of fly paper strung throughout the greenhouse to catch insects, the number and type of insects caught are then manually counted and recorded. This process is required to know which insects need to be introduced to ensure that there is not an infestation that could be harmful to the crops or the pollinating insects. By using 'OpenCV', a python computer vision library, images of the fly paper can be passed to the cloud where the number of insects can be automatically counted and recorded.

Data Analysis

Data that has been imported and stored in the AWS system can be analysed using AWS internal tools or through the setup of external tools. Data can then be aggregated over time, allowing for optimizations to be found, or trends in yield affecting issues to be discovered quickly and effectively responded to.
